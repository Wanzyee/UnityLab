Detailed Description

Create UnityEngine.Material assets for each source UnityEngine.Texture selected.

Open utility window by menu "Assets/Create/Textured Materials...". New assets named as source textures and saved in the same folder. Cancel operation if original named material is existed for safety. Able to edit material source in utility window.
