Detailed Description

UnityEngine.PropertyAttribute to multi-select flag mask enum by drop-down menu.

Only for int enum, within 31 values at most. Better to define enum in order begin from 1, then 2, 4, 8 and so on power of 2. Unless with a specific purpose to integrate, e.g., UnityEngine.RigidbodyConstraints. Note, 'coz the flag system using, 0 always means nothing, -1 as everything.

Example enum like:

[Flags]
public enum Guest:int{
ONE = 1, // or (1 << 0)
TWO = 2, // or (1 << 1)
LOL = 4, // or (1 << 2)
};
